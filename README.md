# 🎵 Spotify Clone  

A simple **Spotify Clone** built using **HTML, CSS, and JavaScript**, replicating the core UI and basic music playback features of Spotify.  
This project was created for learning front-end development concepts and improving UI/UX design skills.  

---

## 🚀 Features
- 🎧 Play, Pause, and Navigate between songs  
- ⏱️ Seekbar and Timer functionality  
- 📂 Organized Music Library  
- 🎨 Fully Responsive UI (Mobile + Desktop)  
- 🔊 Volume Controls  

---

## 🛠️ Tech Stack
- **HTML** – Structure  
- **CSS** – Styling & Layout  
- **JavaScript** – Functionality & Interactions
